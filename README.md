# Mobile-Sales-PowerBI-Dashboard
Power BI dashboard for analyzing mobile sales performance


This project showcases an interactive Mobile Sales Dashboard built using Microsoft Power BI.
The dashboard provides end-to-end insights into mobile sales performance across brands, models, cities, payment methods, and time periods. 
It is designed to help stakeholders quickly understand sales trends, customer behavior, and key business metrics.


# Objectives
1. Analyze overall mobile sales performance.
2. Track MTD (Month-to-Date) sales trends.
3.Compare sales across cities, brands, and mobile models.
4. Understand customer preferences through payment methods & ratings.
5.Enable data-driven decision-making with interactive filters.
